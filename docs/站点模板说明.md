# 站点模板说明

## 重点说明

**模板参数的数组里，如果需要替换参数名，顺序必须保持一致**

例如

```json
{
  "loginParams": ["username", "password"]
}
```

如果`username`替换成`phone`，`loginParams`必须为`["phone", "password"]`。如果变成`["password", "phone"]`，那接单就会出错。
